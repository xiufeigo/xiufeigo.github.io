Windows-Vim安装教程

Author: xiufeigo Date: marzo 21, 2020  19:07:09

上一篇《C语言从入门到入土》中小肥狗使用了Vim编辑器写代码，有有一些小伙伴们在看完教程后跑过来问小肥狗Windows下要怎么装Vim，甚至扬言说不给教程就要取走小肥狗的狗头。于是乎，小肥狗想着好(小)人(命)做(要)到(紧)底(啊)，写了这篇教程来教大家怎么在Windows下使用Vim敲(去)代(装)码(X)。

首先第一步当然是去这里下载最新版的Vim。进入界面后你会看到一堆又一堆的英文。不要慌，不懂英文没关系，左边有一个绿绿的Download在那里，点一下就能进入到下载界面啦。

点进去之后再点击 “PC:MS-DOS and MS-Windows” 这个选项跳转到另一个界面，然后再那个界面点击 “vim-win32-installer” 就可以下载vim的安装包啦！


下载完安装包后双击 “gvim82.exe” 打开安装界面，然后一路点next直到安装完毕。安装成功后桌面会出现一堆vim的快捷方式。

然后呢直接运行这些快捷方式？不不不，我们是要在CMD/Powershell运行才够装X，所以，当然是要在CMD里先来一句vim -v啦！，如果出现下图的界面，那么恭喜你，你的Vim安装完啦！

如果不是，请接着往下看:

一般情况下出现错误都是长这样的:
'vim' 不是内部或外部命令，也不是可运行的程序或批处理文件。

这时候我们只需要把Vim目录添加到系统环境变量中去就可以了。

首先先右键计算机 点 “属性”。如图:

然后点击 “环境变量” → 点击 “Path” → 点击 “新建”

变量名看你心情给，不过还是推荐老老实实填Vim叭。变量值填Vim的安装路径，如果是默认安装的话。64位系统填:

`C:\Program Files (x86)\vim\vim82`

32位的系统填:

`C:\Program Files (x86)\vim\vim82`

然后再去CMD执行 vim -v 会发现Vim已经能够正常运行了。
