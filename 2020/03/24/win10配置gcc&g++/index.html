win10配置gcc&g++

Author: xiufeigo Date: marzo 24, 2020  8:37:09

在上一期“Windows-Vim安装教程”中教会了大家如何在Win10中使用Vim编辑器，但是新的问题又来了，小伙伴们发现使用Vim写完了代码后，在CMD执行 gcc - o HelloWorld.exe HelloWorld.c 时，编译器会提示 'gcc' 不是内部或外部命令，也不是可运行的程序 或批处理文件。 这时候就有小伙伴懵住了，小肥狗你这不坑我嘛，我怎么没办法用这道命令啊！是不是你给的教程有毛病啊！对于这种情况小肥狗得先谢个罪，毕竟只给了你们如何安装Vim却没有教你们配置C语言代码环境，于是乎，迟到了n年的这一片“Win10配置gcc&g++”就闪亮登场啦~
好了，废话不多说，再说命都没了~马上步入教程内容

###方法一
安装gcc的话有两种方法，一种是直接将其他自带gcc的软件添加到path中从而可以在CMD调用gcc，这样的软件有 “Dev C++” 等等。如果用这种方法的话直接跳到path环节将路径改成 “Dev C++” 的安装目录下的 “MinGW64” 下的 “bin”。例如默认安装路径就填：
C:\Program Files (x86)\Dev-Cpp\MinGW64\bin

###方法二

第二种方法的话就是使用mingw包来安装gcc等组件。安装gcc&g++的第一步当然是要去官网下载安装包啊\！（点我传送）（tips：由于官网是国外的，所以跳转速度和下载速度感人）
进入安装网站后我们点击右上角的“Downloads”

点完之后在弹出的窗口里下拉 点击 “MinGW Installation Manager (mingw-get)”，如图：

然后再在接下来的窗口里继续下拉，点击 “MinGW-Get Version 0.6.3 (Beta)”，如图

然后再一次在弹出来的窗口里下拉，点击 “mingw-get-setup.exe”，如图

其实，我可以给你们个下载直链的，啊哈哈哈哈哈哈哈哈哈哈，皮这一下很开心~ 链接
下载完了之后肯定是要安装吖直接放图叭



进度条跑完后长这样，点Continue就行啦！

安装完成后会打开主界面

然后关掉这个界面，跟上期教程一样打开环境变量设置：

在里面填入刚才安装的mingw的安装路径，如果是默认安装的话填入
C:\MinGW\bin
然后保存退出。接下来打开CMD，输入
mingw-get --version
如果没有报错的话，就可以执行下面的指令。如果报错了，检查下path有没有填错或者重启下电脑。
mingw-get install gcc
mingw-get install g++
mingw-get install gdb
mingw-get install mingw32-make
等执行后在cmd输入
gcc -v
如果CMD没有提示错误的话说明安装成功啦可以愉快的写C语言啦
